URL Safety Checker Browser Extension

Real-time URL Safety Protection

Our browser extension provides real-time protection as you browse the web. It analyzes URLs before you visit them and warns you about potentially unsafe websites.

URL Safety Browser Extension

Key Features:

  • Real-time URL checking - Automatically scans URLs before you visit them
  • Safety warnings - Shows interstitial warnings for potentially unsafe sites
  • Link scanning - Highlights potentially unsafe links on web pages
  • Detailed analysis - Provides comprehensive safety information about URLs

Installation Instructions

Note: Our extension is currently in development and will be available in browser stores soon.

Manual Installation

  1. Step 1 Download Extension Download and unzip the extension package
  2. Step 2 Open your browser's extension page:
    • Chrome: Navigate to chrome://extensions
    • Firefox: Navigate to about:addons
    • Edge: Navigate to edge://extensions
  3. Step 3 Enable "Developer mode" (usually a toggle in the top-right corner)
  4. Step 4 Click "Load unpacked" (Chrome/Edge) or "Load Temporary Add-on" (Firefox) and select the extracted extension folder
  5. Step 5 The extension should now appear in your browser toolbar

Using the Extension

Basic Usage:

  • Icon Click: Click the extension icon to check the safety of the current page
  • Automatic Scanning: The extension automatically scans URLs before you visit them
  • Warning Screens: For potentially unsafe sites, a warning screen will appear before loading the page
  • Link Highlighting: Unsafe links on web pages will be highlighted with a red border and warning icon

Configuration:

Right-click the extension icon and select "Options" to configure the extension settings:

  • Protection Level: Adjust the sensitivity of the URL safety checks
  • Notifications: Configure notification preferences
  • Whitelist: Add trusted domains to the whitelist
